# NeetCode 150 – Interview Prep Materials

Materials for 5–10 minute discussion warm-ups and rapid interview prep. Based on the [NeetCode 150](https://neetcode.io/practice) list (Blind 75 + 75 more). **Reference only – not published on website.**

## Files

| File | Purpose |
|------|---------|
| **questions.md** | All 150 problems with descriptions and examples |
| **worksheet.md** | 5-line algorithm summary + complexity per problem (60 most asked). Use for 5–10 min discussion. |
| **answers.md** | Python solutions for 36 key problems |

## How to Use (5–10 Min Discussion)

1. **Pick a problem** from the worksheet (or questions list).
2. **Fill in the 5 lines** — describe the algorithm in 5 bullet points.
3. **Fill in complexity** — Time O(?), Space O(?).
4. **Check answers** — Compare with the answer section at the bottom of each worksheet problem.
5. **Discuss** — Explain the approach aloud in under 2 minutes.
